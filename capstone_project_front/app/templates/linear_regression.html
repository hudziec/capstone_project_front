Linear Regression


This assignment is to explain linear regression in words and terms that someone not specialized in a specific field can understand and using my own words.


Linear regression can be divided into two concepts.


First, is simple linear regression. Mathematically, the relationship between two or more variables. Simple linear regression only works with two variables, an independent variable and a dependent variable. The dependent variable is always on the y axis in a given graph. This is the only one variable that can be explained by the data we have. Since this is the only data we have, the only calculation we can make from it is a mean. The mean, is best represented on a graph by the best fit line. Additionally, given one variable, the mean is the best predictor for estimating future data.


Given a graph with a best fit line and one variable of data on it, how well does it fit? The answer is goodness of fit. This is the calculation between our data points and the distance from the best fit line. This distance is what we call the residual. The residuals for above and below our best fit line will always equal zero. Additional insight can be drawn from squaring the residuals. One, squaring the residuals makes them all positive. Two, it emphasizes the larger deviations. This exaggerates the points that are further away from our best fit line. Three, if we add the sum of the squared errors we get SSE. Finding SSE is a key statistical piece of information. This is because the goal of simple linear regression is to create a linear model that minimizes the sum of squares of the residuals or the SSE. When we add additional independent variables this will either drag down or up our SSE. When our added data does this, we can determine how good that line fits the data by comparing the two variables. When we say a linear regression model is good we are saying that it reduces the square residuals. So in essence, simple linear regression is really a comparison of two models. The first model is where only the mean exists of the dependent variable. The second model is where we introduce independent variables and deduce the regression line or best fit line. The difference between the mean line and the new regression or best fit line is our measure between the two models. Simple linear regression is engineered to find the best fit line through the data that minimizes SSE.


Our second concept in linear regression is multiple regression. Multiple regression is an extension of simple linear regression. That is to say, in simple regression we have a one-to-one relationship between an dependent variable and an independent variable. The independent variable is used to explain the variation in that dependent variable or make predictions about that dependent variable. In multiple regression we have a many to one relationship. We still have a dependent variable but we have multiple independent variables being used to analyze or predict the variation of the dependent variable. Adding multiple variables to a multiple regression model can in fact make things worse, or increase our SSE. This is called overfitting. Another factor in multiple regression is that the independent variables that we are adding to the model may also have a relationship with each other. This is called multicollinearity. The perfect multi-regression model would have multiple independent variables with no relationships to each other and only having a relationship with the dependent variable. The fact is that some independent variables are better at predicting the dependent variable than others.

What is K-Folds Cross Validation?

  • K-fold cross-validation takes the idea of a single train/test split and expands it to multiple tests across different train/test splits of your data.
  • K-fold cross-validation builds K models one for each train/test pair and evaluates those models on each respective test set.
  • The K-fold cross-validation scores appear to be in proximity to out model score indicating that our data is regular and redundant thus giving us confidence in its accuracy and precision.
